Compilation du projet (dans ce repertoire)

Prerequis :
- Java 11 installe et disponible dans le PATH
- Maven installe et disponible dans le PATH

Commandes de verification :Créer un petit redame
java -version
mvn -version

Commande recommandee de compilation/package (genere le .jar et le .exe dans target/) :
mvn clean package

Commande alternative (si vous voulez aussi installer l'artefact dans le depot Maven local) :
mvn clean install

Precisions sur l'artefact :
- Un "artefact" Maven est le livrable produit par la compilation du projet.
- Dans ce projet, l'artefact principal est le JAR : target/winibw-synchro-scripts-client-2.0.3.jar
- Le build genere aussi un executable Windows : target/winibw-synchro-scripts-client-2.0.3.exe
- Avec "mvn clean install", le JAR est en plus copie dans le depot Maven local
  (par defaut : ~/.m2/repository/fr/abes/winibw/winibw-synchro-scripts-client/2.0.3/)
